Important This section of the Office Business Scorecard Manager Web Part Help is written for the business analyst or information technology administrator who creates the Windows SharePoint Services-based or SharePoint Portal Server 2003-based Web page with Business Scorecard Manager Server Web Parts, and who modifies the functionality available on that page using Windows SharePoint Services, SharePoint Portal Server 2003, and Microsoft® Office Business Scorecard Builder.

To create a Windows SharePoint Services-based or SharePoint Portal Server 2003-based Web page with Business Scorecard Manager Server Web Parts, you must complete the following tasks after Business Scorecard Manager Server Web Parts have been installed:

  1. Add Business Scorecard Manager Server Web Parts to a Windows SharePoint Services-based or SharePoint Portal Server 2003-based Web page.

  2. Configure Business Scorecard Manager Server Web parts.

  3. Secure the Windows SharePoint Services-based or SharePoint Portal Server 2003-based Web page against unauthorized access.

For more detailed information about each task, see the appropriate topics under See Also, and see the Microsoft® Office Business Scorecard Manager 2005 Planning and Administration Guide.

Troubleshooting

In addition to setting up and configuring a Windows SharePoint Services-based or SharePoint Portal Server 2003-based Web page with Business Scorecard Manager Server Web Parts, you might have to troubleshoot issues that users of the page encounter in the course of using the scorecard. The table that follows describes common issues that users of the page might encounter, with a solution for each issue.

Issue Solution

Annotations are not available in the Office Scorecard View Web Part, and the Enable Annotations button does not appear on the Office Scorecard View toolbar.

Enable annotations on the scorecard server and in that scorecard view in Microsoft Office Business Scorecard Builder. For information about enabling annotations in Business Scorecard Builder, see Office Business Scorecard Builder Help.

An Office Scorecard View Web Part or Office Report View Web Part is not wide enough to show the entire toolbar.

Increase the width of the view in Business Scorecard Builder. For information about configuring views, see Office Business Scorecard Builder Help.

Alerts are not available in the Office Scorecard View Web Part.

Alerts are created for a scorecard by using Microsoft Office Business Scorecard Builder. They must be enabled on the scorecard server before users can add them to My Alerts in the Office Scorecard View Web Part. For information about alerts, see Office Business Scorecard Builder Help.

Page filters are not available in the Office Scorecard View Web Part.

Enable page filters for the scorecard view in Microsoft Office Business Scorecard Builder. For information about how to do this, see Office Business Scorecard Builder Help.

The Help button on the Office Web Components toolbar does not appear in PivotTable, PivotChart, or spreadsheet report views.

Increase the width of the report view in Microsoft Office Business Scorecard Builder. For information about configuring report views, see Office Business Scorecard Builder Help.
