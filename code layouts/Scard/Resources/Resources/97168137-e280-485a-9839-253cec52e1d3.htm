Edit page filters

Page filters allow you to filter the data in a scorecard view when you want to focus on a subset of the data. For example, in a scorecard for a supermarket chain, you could focus on sales of food as opposed to sales of cleaning products or sales of paper products.

If no Edit Page Filters button is displayed on the Office Scorecard View toolbar, this page filtering functionality is not enabled for your scorecard view.

To edit page filters in an Office Scorecard View Web Part

  1. On the Office Scorecard View toolbar, click Edit Page Filters.

    The name of each dimension for the scorecard is then displayed below the toolbar together with a dimension member box and an ellipsis (…) button.

  2. Click the ellipsis (…) button next to the dimension for which you want to change the page filter settings.

  3. In the Microsoft Business Scorecard Manager dialog box, browse to the dimension member that you want to use to filter data in your scorecard, and then click Save.
