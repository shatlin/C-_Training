Welcome to Office Business Scorecard Manager Web Part Help. This Help contains information about the Office Scorecard View Web Part and the Office Report View Web Part for Microsoft Office Business Scorecard Manager 2005. Web Parts provide the functionality for displaying business scorecards on a Windows SharePoint Services-based or SharePoint Portal Server 2003-based Web site and displaying various graphical representations of data from the scorecards.

Business Scorecard Manager Web Part Help addresses two types of user:

  • Users who view and work with scorecard views and report views on Windows SharePoint Services-based or SharePoint Portal Server 2003-based Web sites should refer to the Office Scorecard View Web Part help topics or the Office Report View Web Part help topics.

  • Business Scorecard Manager administrators or others who create or assemble Windows SharePoint Services-based or SharePoint Portal Server 2003-based Web pages by adding and configuring Office Scorecard View Web Parts and Office Report View Web Parts should refer to the Managing Web Parts help topics. These users should also refer to the documentation in the Office Business Scorecard Builder Help and the Microsoft® Office Business Scorecard Manager 2005 Planning and Administration Guide.

Office Scorecard View

The Office Scorecard View Web Part enables you to view scorecard views that have been created in your organization. You can also interact with scorecards by expanding and collapsing rows in the scorecard, filtering the scorecard, and annotating the scorecard. Your organization determines which scorecards you are able to view.

A scorecard view is a hierarchical arrangement of the key performance indicators (KPIs) and objectives that are contained in a scorecard. These KPIs and objectives are the metrics chosen by your organization for measuring progress with respect to defined business goals.

Office Report View

The Office Report View Web Part is connected to an Office Scorecard View Web Part, and usually presents a different view of some or all of the data in a scorecard view. Actions in the Office Scorecard View Web Part can change the data displayed in the Office Report View Web Part. The Office Report View Web Part allows you to choose and view different types of report formats, such as a PivotChart report or a spreadsheet report. The different report format options show the progress the organization is making toward specific goals or show data at a greater level of detail than is displayed in a scorecard view. The report views can be in the form of PivotTable reports, PivotChart reports, spreadsheets, Web pages, scorecards, or reports that are created by using Microsoft SQL Server Reporting Services. The Report View Web Part can also be used to display custom-made report views created by your organization.

Note:
To view and interact with PivotTable report views, PivotChart report views, and spreadsheet report views, you must have Office 2003 Web Components installed on your computer.
