Filter rows by indicator status
See Also






In the Office Scorecard View Web Part, you can temporarily remove some key performance indicators (KPIs) and objectives in a scorecard view by using the Filter Scores feature. For example, temporarily removing from a scorecard view those KPIs and objectives for which the actual values are on target or moderately on target focuses attention on the KPIs and objectives that need the most improvement.

To filter rows by indicator status

  1. On the Office Scorecard View toolbar, click the Filter Status button.

  2. In the Only show rows with indicators list, select the type of indicator that you want to display in the scorecard view.
