Office Report View Web Part
See Also






In an Office Report View Web Part, a component of Microsoft® Office Business Scorecard Manager 2005, you can see and interact with views of data that are either derived from the scorecard view displayed in the Office Scorecard View Web Part, or data that comes from other data sources. Business Scorecard Manager report views are presentations of business data in the form of PivotTable reports, PivotChart reports, spreadsheets, scorecards, Web pages, or SQL Server Reporting Services reports. Additionally, the Office Report View Web Part can display a custom-made report view created by your organization.

Note:
To view and interact with a PivotTable report, PivotChart report, or spreadsheet, you must have Office 2003 Web Components installed on your computer. If Office 2003 Web Components is not installed on your computer, a red x appears when you attempt to view a PivotTable report, PivotChart report, or a spreadsheet.

If an Office Report View Web Part is providing a detailed view of data that you select in an Office Scorecard View Web Part, it displays different report views as you change your selection in the business scorecard. For example, detailed sales data for specific sales regions might be shown as sales per fiscal year and fiscal quarter for each region in a PivotTable report view. The contents of the PivotTable report would change each time you changed your fiscal year selection in Scorecard View.

If an Office Report View Web Part is providing supplemental data from outside data sources, that view does not change as you change your selection in the connected Office Scorecard View Web Part. For example, a Web page report view might display additional information about the sales strategy for an organization in Web page format, and the content of this page would remain the same as you change your selections in the connected Office Scorecard View Web Part.
