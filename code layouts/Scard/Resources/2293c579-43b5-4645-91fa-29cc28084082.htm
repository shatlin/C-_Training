Term Definition

ADOMD.NET

A standard .NET data provider that provides client computer access to multidimensional data sources, such as Microsoft SQL Server 2000 Analysis Services. ADOMD.NET stands for ActiveX Data Objects MultiDimensional.NET.

Balanced Scorecard

A scorecard based on the principles of the balanced scorecard collaborative that is used to plan strategy by measuring performance on four or more perspectives with their attendant objectives and key performance indicators (KPIs).

category

An optional feature of Microsoft Business Scorecard Builder that enables you to easily manage scorecards, data sources, indicators, and key performance indicators (KPIs) in the Business Scorecard Manager database by giving you a hierarchical view of this data.

configured views

Hierarchical arrangements of the KPIs and objectives contained in a scorecard, which can also be configured to include additional data from scorecard data sources.

cube

A multidimensional data structure defined by dimensions, measures, hierarchies, and levels. A cube used to create a scorecard generally corresponds to a functional area of an organization and is designed to meet the analysis needs of that functional area.

data source

A source of business data used by Business Scorecard Manager to create scorecards. Data sources include online analytical processing (OLAP) cubes, relational databases, or any other Open Database Connectivity (ODBC) or ADOMD.NET data source.

dimension

A categorically consistent view of data, in which all dimension members belong together as a group. Dimensions break into hierarchies that consist of levels. Each child level of a dimension contains data in greater detail than its parent. For example, a dimension defined by store location might contain the following levels: continent, country, region, state/province, city, and store number.

element

Any one of the individual components of a scorecard, such as key performance indicators (KPI), data sources, and indicators.

fixed value

A user-entered value for a key performance indicator (KPI), in contrast to target and actual values, which are retrieved from a data source.

general scorecard

The type of scorecard used in Business Scorecard Manager that contains key performance indicators (KPIs) and groups of KPIs called objectives that are arranged in levels to measure strategy and performance.

indicator

A graphical status symbol that represents where the value of a key performance indicators (KPI) is in relation to the target value. Indicators are represented by using status images and colors (indicator band colors). Users that configure the indicator designate a color for each indicator band.

Internet Protocol security (IPSec)

A set of industry-standard, cryptography-based protection services and protocols. IPSec protects all protocols in the TCP/IP protocol suite and Internet communications by using Layer Two Tunneling Protocol (L2TP).

IPSec

See definition for: Internet Protocol security

key performance indicator (KPI)

An explicit and measurable value taken directly from a data source. KPIs are used to measure performance in a specific area, such as revenue per customer.

KPI

See definition for: key performance indicator

Business Scorecard Builder

A client application that you use to create and manage scorecards and to create views of scorecards on a Microsoft SharePoint site.

Business Scorecard Manager database

The database that stores the Business Scorecard Manager metadata, which consists of user permissions and settings for scorecard elements.

Business Scorecard Manager Server

A computer or group of computers that host the Business Scorecard Manager database and the Business Scorecard Manager Web Service.

Business Scorecard Manager Views

A visual presentation of scorecard elements, defined in Business Scorecard Builder, and presented in SharePoint Products and Technologies.

Business Scorecard Manager Web Parts

Provide Business Scorecard Manager Views defined in Business Scorecard Builder to users of SharePoint Products and Technologies.

Business Scorecard Manager Web service

A collection of application programming interfaces (APIs) that provide the core functionality of Business Scorecard Manager, connecting Business Scorecard Builder to the Business Scorecard Manager database, Business Scorecard Manager Web Parts, and any data sources.

MDX formula

The online analytical processing (OLAP) equivalent of an SQL query. MDX stands for multidimensional expression.

measure

A value that represents an important metric in an organization. In Business Scorecard Manager, key performance indicators (KPIs) are measures used to monitor progress towards specific performance goals.

metadata

Data about data. The metadata for a file can include its title, subject, author, and size. In Business Scorecard Manager, the metadata includes user permissions and settings for scorecard elements.

Microsoft XML (MSXML)

A Java-based XML parser from Microsoft for building high-performance XML-based applications that provide a high degree of interoperability with other applications that adhere to the XML 1.0 standard.

MSXML

See definition for: Microsoft XML

objective

A group of key performance indicators (KPIs) that correspond to an organization's functional areas or strategic initiatives.

ODBC

See definition for: Open Database Connectivity

Office Web Components

A set of controls that you use to create data analysis and reporting solutions, both in a Web browser and in traditional programming environments.

OLAP

See definition for: online analytical processing

online analytical processing

A technology that uses multidimensional structures to provide rapid access to data for analysis.

Open Database Connectivity

An application programming interface (API) that enables database applications to access data from a variety of existing data sources.

OWC

See definition for: Office Web Components

PivotChart report

Provides a graphical representation of the data in a PivotTable report.

PivotTable report

An interactive, crosstabulated Microsoft Excel report that summarizes and analyzes data, such as database records, from various sources, including ones that are external to Excel.

relational database

A database or database management system that stores information in tables. In conducting searches, a relational database matches information from a field in one table with information in a corresponding field of another table to produce a third table that combines requested data from both tables.

report views

A multifunctional perspective on scorecard data and data from other sources that take the form of PivotTable reports, Pivot Chart reports, spreadsheets, Web Pages, and SQL reports.

rollup

The calculated value of an item based on a weighted average of the value of all its child entries.

scorecard

A hierarchical collection of key performance indicators (KPIs) and objectives used to measure multiple facets of an organization's performance.

server farm

A centralized grouping of network servers maintained by an enterprise or, often, an Internet service provider (ISP). A server farm provides a network with load balancing, scalability, and fault tolerance. Individual servers may be connected in such a way that they appear to represent a single resource.

SharePoint Portal Server

Capitalizes on Windows SharePoint Services by using SharePoint sites to create portal pages for people, information, and organizations. While these sites are specific to SharePoint Portal Server 2003, they use technologies such as Web Parts and SharePoint document libraries that are delivered by the Windows SharePoint Services platform. This integration greatly reduces the time and cost associated with development, training, and support.

SharePoint Products and Technologies

Includes SharePoint Portal Server and Windows SharePoint Services.

target value

The value for the goal of a key performance indicator (KPI).

value

Used to qualify a key performance indicator (KPI). The KPI values are current, previous, and target.

view

A presentation of scorecard data that users can look at and potentially manipulate to see other data. Views are created by using Business Scorecard Manager Web Parts on a SharePoint site that connect to scorecards created in Business Scorecard Builder.

Web Part

A modular unit of information that consists of a title bar, a frame, and content. It is a basic building block of a Web Part page on a SharePoint site.

Web Part description file (.dwp)

An XML file that defines default property settings for installing a Web Part, including references to any other files required to run and display the part.

weight

The value of a scorecard element in relation to other values. For example, you might have an objective with two key performance indicators (KPIs), the first with a weight of one and the second with a weight of three. This means that the second KPI is three times more important than the first, and this weighted relationship is part of the calculation when the values of these KPIs are rolled up to derive the values of their parent objective.

Windows SharePoint Services

The engine for creating Web sites that enable information sharing and document collaboration, thereby helping to increase individual and team productivity.

workspace

A file that stores the elements (key performance indicators [KPIs], data sources, and indicators) on which scorecards are based. A workspace file has a .bsw file name extension.
