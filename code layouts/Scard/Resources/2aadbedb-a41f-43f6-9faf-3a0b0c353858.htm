Office Scorecard View Web Part
See Also






The Office Scorecard View Web Part in Microsoft® Office Business Scorecard Manager 2005 allows you to see scorecard views that have been created by your organization. The Office Scorecard View Web Part displays a hierarchical arrangement of the key performance indicators (KPIs) and objectives that are contained in whatever scorecard you have chosen to view.

The Office Scorecard View Web Part provides visual indicators that show progress against performance targets at the KPI level and at a higher level of collections of KPIs called objectives. KPI-level and objective-level indicators also can be rolled up to provide a scorecard-level indicator.

In the Office Scorecard View Web Part, you can do the following:

  • Either edit the page filter or use filter mode to limit the data that is displayed.

  • Filter out the KPIs and objectives with indicator values above a given level. This allows you to focus on those KPIs and objectives that are not performing well against target values.

  • Subscribe to an alert, so that you are notified with an e-mail message when certain aspects of the scorecard view change.

  • Add an annotation to provide additional information about a cell in a scorecard view.

  • Expand and collapse rows.
