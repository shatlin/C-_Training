Add annotations to an Office Scorecard View Web Part
See Also






The annotations functionality makes it possible for you to add comments and information to the cells of a scorecard view in an Office Scorecard View Web Part on your Web page. For example, you might want to make a notation in a cell with store sales numbers, indicating that bad sales at this time were caused by poor weather. Other users can then see the comment you added to this cell when they use the Office Scorecard View Web Part.

If there is no Enable Annotation button on the Office Scorecard View toolbar, contact your administrator.

To add an annotation to a cell in a scorecard view

  1. In either SharePoint Portal Server or Windows SharePoint Services, on the Office Scorecard View toolbar, click Enable Annotation.

  2. Click the cell where you want to add an annotation.

  3. In the Microsoft Business Scorecard Manager dialog box, click Add Comment.

  4. In the Enter Title box, type the title of the comment.

  5. In the Enter Comment box, type the comment.

  6. Click Save.

    Repeat steps 3-6 to add more comments to the annotation.

  7. Click Close.
Note:
Cells that contain an annotation are marked with a red triangle.
