Add an Office Scorecard View Web Part to a Web page
See Also






To add a Microsoft Office Business Scorecard Manager Server Office Scorecard View Web Part to a Windows SharePoint Services-based or SharePoint Portal Server 2003-based Web page, use the following instructions.

To add a Scorecard Web Part to a Web page

  1. On the SharePoint Portal Server 2003-based or Windows SharePoint Services-based scorecard Web page, click Modify Shared Page.

    If the Modify Shared Page link does not appear on the Web page, you must first click Edit Page.

  2. Click Design This Page.

  3. Click Modify Shared Page again, point to Add Web Parts, and then click Browse.

    If the Modify Shared Page link does not appear on the Web page, you must first click Edit Page.

  4. In the Add Web Parts navigation bar, under Browse, click Virtual Server Gallery.

  5. Under Web Part List, drag Office Scorecard View to the Left or Right zone on the Web page.

  6. In the Business Scorecard navigation bar, select a scorecard in the Select a Scorecard list.

  7. In the Select a View list, select the scorecard view that you want to display, and then click OK.
Note:
To display a different scorecard view in the Office Scorecard View Web Part, click the arrow in the upper-right corner of the Office Scorecard View Web Part. Then click Modify Shared Web Part to select a different scorecard view from the Select a View list.
