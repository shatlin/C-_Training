Filter scorecard views
See Also






You can filter scorecard views to display or hide data using the following features:

  • Page filters: Page filters allow you to filter the data in the Office Scorecard View Web Part when you want to focus on a subset of the data. For example, in a scorecard for a supermarket chain, you could focus on sales of food as opposed to sales of cleaning products or sales of paper products.

  • Filter mode: You can use filter mode to sort or filter the rows that are displayed in a scorecard view according to a variety of criteria. This allows you to select the type of information and the level of detail that is most relevant for evaluating specific aspects of performance goals or strategies. For example, you might want to use filter mode to display only rows for the grocery stores with the highest annual sales.

  • Filter rows by indicator status: In a scorecard view, you can temporarily remove some key performance indicators (KPIs) and objectives from a scorecard view by using the Filter Scores feature.

For more information on each of these features, see the topics listed in See Also.
