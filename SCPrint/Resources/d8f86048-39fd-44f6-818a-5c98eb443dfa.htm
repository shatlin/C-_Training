Use Filter Mode and View Mode
See Also






You can use filter mode to sort or filter the rows that are displayed in the Office Scorecard View Web Part according to a variety of criteria. This allows you to select the type of information and the level of detail that is most relevant for evaluating specific aspects of performance against goals or strategies. For example, you might want to use filter mode to display data only for the grocery stores with the highest annual sales.

View mode will always display all of the key performance indicators (KPIs) and objectives in the scorecard view. But if you switch back and forth between view mode and filter mode, you will lose the filter settings and have to reapply them each time.

To switch between filter mode and view mode

  • On the Office Scorecard View toolbar, click Filter Mode to switch to filter mode, or click View Mode to switch to view mode.

To filter a scorecard view in filter mode

  1. On the Office Scorecard View toolbar, click Filter Mode .

    A drop-down list appears under each column heading that provides a variety of options for sorting the data in the scorecard.

  2. In the drop-down list under each column heading, you can do any of the following:

    • Click Sort Ascending to sort the scorecard rows according to ascending values.

    • Click Sort Descending to sort the scorecard rows according to descending values.

    • Click (All) to show all scorecard rows.

      You can use this option to undo the (Top 10) and (Custom) options.

    • Click (Top 10) to show the scorecard rows with the highest or lowest values.

      In the Numerical Ranking - Web Page Dialog box, choose either Top or Bottom.

      In the text box, type the number of rows that you want to display.

      Choose Items to display a fixed number of rows, or click Percentage to display a specified percentage of rows.

    • Click (Custom) to show a customized collection of rows with values that meet specific criteria.

      In the Custom Filter - Web Page Dialog box, select one or more operators in the drop-down lists.

      In the text boxes, type one or more numerical values. For example, you could choose is greater than for the operator, and then enter a numerical value of 1000 if you wanted to show only grocery stores with sales over $1000.

  3. Click OK.
