Add an Office Report View Web Part to a Web page
See Also






To add a Microsoft Office Business Scorecard Manager Server Office Report View Web Part to a Windows SharePoint Services-based or SharePoint Portal Server 2003-based Web page, use the following instructions.

You can display the report views assigned to any of the report view groups, and you can also connect the Office Report View Web Part to the Scorecard View Web Part, so that the Office Report View Web Part will display the data that is related to KPIs, objectives, or cells that users select in the Office Scorecard View Web Part.

Report view groups allow you to select a report view or set of report views and make those views available in the Office Report View Web Part. For information about assigning report views to report view groups, administrators should see Office Business Scorecard Builder Help.

To add an Office Report View Web Part to a Web page

  1. On the SharePoint Portal Server 2003-based or Windows SharePoint Services-based scorecard Web page, click Modify Shared Page.

    If the Modify Shared Page link does not appear on the Web page, you must first click Edit Page.

  2. Click Design This Page.

  3. Click Modify Shared Page again, point to Add Web Parts, and then click Browse.

  4. In the Add Web Parts navigation bar, under Browse, click Virtual Server Gallery.

  5. Under Web Part List, drag Office Report View to one of the zones on the Web Part page.

  6. In Office Report View, click the arrow, and then click Modify Shared Web Part.

  7. Under Report View Menu Items, select the check boxes of the report view groups that you want to display. The Report View Menu Items list indicates how many scorecard report views and KPI report views are assigned to each report view group.

    Note:
    In Business Scorecard Builder Help, we recommend that the person who creates a report view set the height and width for the report view in Business Scorecard Builder. Therefore, we recommend that you use the following settings on the SharePoint site in the Report View Menu Items dialog box. Expand Appearance, under Height, select the No. Adjust height to fit zone option; under Width, select the No. Adjust width to fit zone option. In the Frame Style box, select None to remove white space in the Web Part.

  8. To include scorecard report views in a drop-down list in the Office Report View Web Part even when a KPI is selected in the connected Office Scorecard View Web Part, select Always show scorecard Report Views.

  9. Click OK.

To connect an Office Report View Web Part to a Scorecard View Web Part

  1. On the SharePoint Portal Server 2003-based or Windows SharePoint Services-based scorecard Web page, click Modify Shared Page.

    If the Modify Shared Page link does not appear on the Web page, you must first click Edit Page.

  2. Click Design This Page.

  3. Click the arrow in Office Report View.

  4. Point to Connections, point to Consumes view data from, and then click Business Scorecard.

  5. Click OK.
