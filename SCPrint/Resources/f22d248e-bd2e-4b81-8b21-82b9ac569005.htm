Use a scorecard report view
See Also






The scorecard report view displays a scorecard view in the Office Report View Web Part. The scorecard view it displays is associated with a scorecard, key performance indicator (KPI), or objective that you select in the Office Scorecard View Web Part.

For example, a scorecard Web site might include an Office Scorecard View Web Part that shows a scorecard view of grocery store sales, budgets, and profits for stores in each city in Washington State, and an Office Report View Web Part that shows a scorecard view of budgets for each type of grocery store in Washington State. The scorecard might also include a PivotTable report view and a PivotChart report view, which are included in the Report View Web Part. When you click the budget KPI in the Office Scorecard View Web Part, the scorecard report view appears in the Office Report View Web Part, showing the budget for each grocery store in Washington State. When you click sales or profit KPIs in the Office Scorecard View Web Part, the PivotChart report view or PivotTable report view appears in the Office Report View Web Part.

Scorecard report views support the full range of scorecard view features. For more information about these features, see the topics listed under See Also.

To toggle between a scorecard report view and other available report views

  • In the Scorecard View Web Part, do one of the following:

    • To display a scorecard report view, click a scorecard, KPI, or objective associated with it.

    • To display another available report view, click a scorecard, KPI, or objective associated with it.
